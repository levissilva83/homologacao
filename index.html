Segurança dos Dados

Seus dados blindados.
Sem exceção.

🛡️
🔑
Token de Sessão Criptografado
Cada login gera um token único. Sem token válido, nenhuma requisição é aceita. Proteção em todas as camadas.
⏱️
Expiração Automática de Sessão
Sessão expira em 24h automaticamente. Token vencido = redirecionamento imediato para login.
📧
Autenticação E-mail + Senha
Acesso vinculado ao e-mail. Recuperação com confirmação dupla. Senha mínima de 6 caracteres.
🚫
Bloqueio de Acesso Não Autorizado
Qualquer requisição sem autenticação é bloqueada. Alunos e admins têm escopos separados.
🔒
Troca de Senha com Verificação Dupla
Senha atual obrigatória + confirmação. Impossível alterar sem acesso prévio autenticado.
